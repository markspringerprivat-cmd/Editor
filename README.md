# H5P-ähnliche Editoren

Dieses Paket ist von der Kurswebseite getrennt. Es enthält nur die selbst nachgebauten Editor-/Aktivitätsseiten:

- Interaktives Video
- Course Presentation
- Interactive Book
- Multiple Choice
- Drag the Words
- Drag and Drop

Startdatei: `index.html`

Die Exporte sind HTML-ZIP-Dateien mit eigenen Laufzeitdateien. Es handelt sich nicht um echte `.h5p`-Pakete und nicht um eine Moodle-Gradebook-Anbindung.
